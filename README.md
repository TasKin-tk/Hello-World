# Hello-World

A collection of "Hello, World!" implementations in various programming languages.  

---

## English

### Description

Hello-World is a curated repository of "Hello, World!" programs written in various programming languages.  
This project serves as a quick reference for comparing syntax across different languages.  
Each file contains standalone source code, viewable and comparable directly in any plain-text editor. For runtime environments, please refer to the official documentation of each language.  

### Categories

- **Classic Languages** — C, C++, Java, Python, Go, Rust, C#, JavaScript, TypeScript...  
- **Scripting Languages** — Ruby, PHP, Perl, Lua, R...  
- **Functional Languages** — Haskell, Lisp, Erlang, Elixir, Clojure...  
- **System Languages** — Assembly (x86, ARM, MIPS), Fortran, COBOL, Ada...  
- **Esoteric Languages** — Brainfuck, LOLCODE, Befunge, Whitespace...  
- **Web Languages** — HTML, CSS, Dart...  
- **Others** — SQL, MATLAB, Julia, Swift, Kotlin...  

### Features

- A wide range of programming languages covered  
- Clean, runnable code examples  
- Organized by language category for easy browsing  
- Each file is independent with no external dependencies  
- Cross-platform — works on Windows, Linux, macOS  

### Directory Structure

```
Hello-World/
├── classic/          - C, C++, Java, Python, Go, Rust...
├── scripting/        - Ruby, PHP, Perl, Lua, R...
├── functional/       - Haskell, Lisp, Erlang, Elixir...
├── system/           - Assembly, Fortran, COBOL, Ada...
├── esoteric/         - Brainfuck, LOLCODE, Befunge...
├── web/              - HTML, CSS, Dart...
└── other/            - SQL, MATLAB, Julia, Swift...
```

### Quick Start

Clone the repository and browse the code:
```
git clone https://github.com/TasKin-tk/Hello-World.git
cd Hello-World
ls
```

### Contributing

Contributions are welcome! If you'd like to add a new language:

1. Fork this repository  
2. Create a new directory under the appropriate category  
3. Add your `hello.*` file  
4. Update the language list in this README  
5. Submit a Pull Request  

Please ensure your code runs without errors before submitting.

## 中文

### 简介

Hello-World 是一个收录了多种编程语言 "Hello, World!" 实现的代码仓库。  
本仓库旨在作为快速对比不同语言语法的参考手册。  
每个文件均为独立源代码，可直接在纯文本编辑器中浏览和对比，各语言的运行环境请参考官方文档。  

### 分类

- **经典语言** — C、C++、Java、Python、Go、Rust、C#、JavaScript、TypeScript...  
- **脚本语言** — Ruby、PHP、Perl、Lua、R...  
- **函数式语言** — Haskell、Lisp、Erlang、Elixir、Clojure...  
- **系统语言** — 汇编（x86、ARM、MIPS）、Fortran、COBOL、Ada...  
- **深奥语言** — Brainfuck、LOLCODE、Befunge、Whitespace...  
- **Web 语言** — HTML、CSS、Dart...  
- **其他语言** — SQL、MATLAB、Julia、Swift、Kotlin...  

### 功能特点

- 涵盖多种编程语言  
- 代码简洁清晰，可直接运行  
- 按语言分类存放，便于查找  
- 每个文件独立无依赖  
- 跨平台支持：Windows、Linux、macOS
